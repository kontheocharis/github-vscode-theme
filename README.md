# GitHub's VS Code themes but in grayscale

> **Note**: This fork was made by Claude AI

![GitHub VS Code theme (Grayscale)](./sample.png)

A fork of GitHub's VS Code theme with grayscale backgrounds instead of blue-tinted ones for dark themes.

## Install

1. Download from the [VS Marketplace](https://marketplace.visualstudio.com/items?itemName=kontheocharis.github-vscode-theme-grayscale) or search for "GitHub Theme (Grayscale)" in VS Code.
2. Click on the "Install" button.
3. Then [select a theme](https://code.visualstudio.com/docs/getstarted/themes#_selecting-the-color-theme). Available themes:
    - `GitHub Light Default`
    - `GitHub Light High Contrast`
    - `GitHub Light Colorblind`
    - `GitHub Dark Default (Grayscale)`
    - `GitHub Dark High Contrast (Grayscale)`
    - `GitHub Dark Colorblind (Grayscale)`
    - `GitHub Dark Dimmed (Grayscale)`
